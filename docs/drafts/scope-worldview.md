# 功能探讨: 天衍世界观设定

## 基本信息

- **创建时间**: 2026-02-02
- **关联项目蓝图**: #1（NodeMUD 项目蓝图）
- **前置依赖**: 角色创建系统（Scope 已完成）、Layer 0-4 引擎（全部完成）
- **后续影响**: 出生点地图设计、区域规划、门派系统、主线剧情

## 功能目标

为"人在江湖"游戏构建完整的世界观体系——**天衍世界**。采用跨时代架空 + 多纪元 + 断层纪元的设计，融合金庸（家国情怀）、古龙（人心博弈）、燕云十六声（多方割据）三种调性，结合中国五千年历史文化元素，打造独特的武侠世界。

世界观的确立将直接指导后续的出生点地图、区域蓝图、NPC 势力、主线剧情等所有内容设计。

## 核心设定

### 世界名称：天衍

- **出处**："大衍之数五十"——《易经·系辞》，天道演化之意
- **含义**：天道在衍化，命格在推演，世界在轮转
- **与游戏系统的呼应**：紫微斗数命格系统正是"天衍"——天道推演命运的具象化

### 纪元体系：三纪元

采用**线性纪元 + 断层痕迹**的结构。玩家活动在当世纪，前两个纪元以遗迹、秘籍、传说、古墓的形式存在于世界中。

| 纪元       | 历史灵感                   | 世界观定位                             | 武学特征                         | 持续时间       |
| ---------- | -------------------------- | -------------------------------------- | -------------------------------- | -------------- |
| **太古纪** | 三皇五帝、先秦诸子         | 神话时代，人与天地相通，修炼可通神     | 上古功法，以"道"驭力，近乎仙术   | 不可考         |
| **中古纪** | 汉唐宋，武林鼎盛           | 江湖黄金时代，门派林立，百家争鸣       | 各派武学成型，内外兼修，体系完备 | 约两千年       |
| **当世纪** | 五代/南宋末/明末融合的乱世 | 天裂后的重建时代，旧秩序崩塌，江湖重洗 | 武学断层，功法失传，重新摸索     | 天裂后约三百年 |

#### 太古纪

- 人与天地相通，修炼者可感应天地法则
- 先贤将天地法则刻录于九块石碑之上——**天衍碑**
- 太古纪末年，天地元气自然回落，神话时代终结，进入中古纪
- 太古纪的遗迹和功法是天衍世界武学的终极天花板

#### 中古纪

- 武林黄金时代，十大宗门鼎立，百家争鸣
- 武学体系完备：内功、外功、轻功、暗器、毒术各成派系
- 各大宗门陆续发现太古纪遗留的天衍碑
- 末年因争夺天衍碑引发大劫——天裂

#### 当世纪（玩家所处时代）

- 天裂后三百年，世界在愈合中
- 天地元气稀薄，中古纪的顶尖功法因元气不足无法施展
- 弱势朝廷名义统治中原，各方势力割据
- 中古纪和太古纪的遗迹散落天下，成为探索核心内容

### 天裂之劫：大灾变

**性质**：人祸与天灾的结合

**过程**：

> 中古纪末年，天下武学极盛，各大宗门发现太古纪遗留的"天衍碑"——传说是太古纪先贤记录天地法则的石碑，散落于天衍大陆各处，共九块。各方势力为争夺天衍碑大战数十年（**人祸**），其中三块在混战中被强行催动，引发天地灵脉共振崩裂（**天灾**）。

**后果**：

- **九块天衍碑下落不明**（后续高级玩法 / 主线剧情内容）
- **十大宗门覆灭其七**，仅三门残存
- **天地元气稀薄**，中古纪顶尖功法因元气不足无法施展
- **大量地貌重塑**，城池沉没、山脉升起、海域变陆地（中古纪遗迹成为地下副本）
- **人口锐减**，文明倒退，大量技艺和知识失传

**设计意图**：

- 世界名"天衍"有了实体锚点（天衍碑）
- 九块碑 = 后期高级探索 / 主线内容
- "元气稀薄"解释了当世纪武学弱于中古纪
- 地貌重塑 = 地理可以自由设计，不需严格对应真实中国版图
- 给蓝图系统预留了"遗迹副本"的扩展空间

### 区域详述

| 区域          | 地貌特征         | 主导调性         | 核心内容                           |
| ------------- | ---------------- | ---------------- | ---------------------------------- |
| **北境·霜疆** | 草原、关隘、冰原 | 金庸（家国情怀） | 外族入侵、边关守卫、军旅热血       |
| **中原·腹地** | 平原、旧城、废墟 | 三调性交汇       | 朝廷政治、旧门派遗迹、势力争夺     |
| **东南·江南** | 水乡、湖泊、园林 | 古龙（暗流涌动） | 商帮暗战、情报网络、文人风骨       |
| **东海·岛屿** | 海岛、暗礁、港口 | 古龙（孤独浪客） | 海盗、仙岛传说、散人联盟           |
| **西南·蛮疆** | 深山、密林、溶洞 | 燕云（多方割据） | 苗疆巫蛊、部落自治、独立武学体系   |
| **西域·荒原** | 沙漠、戈壁、古城 | 三调性融合       | 太古纪遗迹密集、丝路商道、密宗武学 |
| **南疆·蛮荒** | 热带丛林、河流   | 燕云（混乱割据） | 多部落混战、生存法则、底层江湖     |

### 调性分布逻辑

- **金庸调性（北境）**：外族压境，家国冲突，"侠之大者为国为民"的大侠叙事
- **古龙调性（江南/东海）**：表面繁华下暗流涌动，神秘组织、暗杀、悬疑，人心博弈
- **燕云调性（西南/南疆）**：多方势力割据，无明确朝廷管控，各帮派部落自治
- **中原/西域**：三种调性交汇融合

## 势力格局

### 朝廷：承天朝

- 天裂后百年混战，一方势力统一中原核心区域，建立承天朝
- 实际控制力仅限**中原腹地**，对边疆和偏远地区只有名义统治
- 内部不稳：文官武将争权、外戚干政、皇位更替频繁
- 对江湖态度暧昧：既想收编武林高手为己用，又忌惮江湖势力

### 八方势力

| 势力          | 区域     | 调性      | 特征                                               |
| ------------- | -------- | --------- | -------------------------------------------------- |
| **承天朝廷**  | 中原     | 金庸+燕云 | 弱势朝廷，有官军但战力一般，靠政治手段维持         |
| **北漠·狼庭** | 北境草原 | 金庸      | 游牧外族政权，虎视中原，对标蒙古/契丹              |
| **残存三门**  | 散布各地 | 金庸      | 天裂中幸存的三个中古纪宗门，底蕴深但衰败中         |
| **江南·暗河** | 江南水乡 | 古龙      | 地下情报/暗杀组织网络，无人知道首领是谁            |
| **东海·散盟** | 东海岛屿 | 古龙      | 海上散人联盟，亦商亦匪，自由不羁                   |
| **西南·百蛮** | 西南蛮疆 | 燕云      | 苗疆部落联盟，有独立的巫蛊武学体系                 |
| **西域·遗民** | 西域荒原 | 三者融合  | 太古纪遗迹守护者后裔，神秘封闭                     |
| **天裂教**    | 隐匿各地 | 古龙      | 信奉天裂是"净化"而非灾难的邪教，暗中收集天衍碑碎片 |

### 残存三门（中古纪幸存宗门）

天裂中十大宗门覆灭其七，仅三门残存，是当世"正道"的象征：

| 宗门       | 位置      | 特征                                   | 对标灵感        |
| ---------- | --------- | -------------------------------------- | --------------- |
| **嵩阳宗** | 中原·嵩山 | 最正统，与朝廷关系密切，但保守僵化     | 少林 + 正统名门 |
| **云岳派** | 西部·高山 | 内功心法传承最完整，但人丁稀少         | 武当 + 隐世门派 |
| **碧澜阁** | 东南·湖畔 | 兼修文武，有大量中古纪典籍，但内部分裂 | 书院 + 峨眉     |

## 出生点设计

### 方案：裂谷中立小镇

天裂后地脉重塑，在中原边缘形成了一条巨大的裂谷地带。裂谷中有一个小镇，因位于各方势力的交界处，反而成了**中立贸易/落脚点**。

**选择理由**：

- 所有出身的角色都有合理理由汇聚于此
- 各方势力 NPC 都可以安排在此（酒馆、商铺、情报点）
- 作为向外辐射到各区域的枢纽
- "裂谷"本身就是天裂的物理痕迹，强化世界观沉浸感

**各出身汇聚理由**：

| 出身     | 来到小镇的理由                               |
| -------- | -------------------------------------------- |
| 世家子弟 | 来此寻找中古纪遗迹线索，追寻家族往昔         |
| 江湖浪子 | 四处游荡到此，裂谷小镇是消息最灵通的地方     |
| 书院学子 | 追踪古籍记载来此考察天裂遗迹                 |
| 边塞军卒 | 退役后南下谋生，裂谷是北境通往中原的必经之路 |
| 山野药童 | 下山采购交易药材，裂谷中有独特的药草         |
| 乞丐流民 | 随人流漂到此，裂谷小镇收容所有人             |

## 探讨记录

### 关键决策过程

1. **世界名称选择**：考虑了"玄黄"（天地玄黄）、"坤舆"（大地雅称）、"天衍"（大衍之数）、"苍梧"（上古神山）、"承渊"（原创）等方案。最终选择"天衍"，因为与紫微斗数命格系统呼应（天道推演命运），且"衍"字暗示世界在纪元中不断演化。

2. **纪元体系**：确认三纪元——太古纪（神话）、中古纪（武林黄金时代）、当世纪（断层乱世）。采用线性纪元 + 断层痕迹结构，旧纪元以遗迹形式存在。

3. **大灾变性质**：确认天裂之劫为人祸（争夺天衍碑）+ 天灾（灵脉共振崩裂）结合。这比纯人祸或纯天灾更有层次感。

4. **调性融合策略**：不选择单一调性，而是用地理分区承载不同调性——北境（金庸·家国）、江南/东海（古龙·暗流）、西南/南疆（燕云·割据）、中原（三调性交汇）。

5. **朝廷设定**：确认当世纪有朝廷（承天朝）但很弱，各方势力不服。这创造了最大的故事张力。

6. **出生点方案**：选择"裂谷中立小镇"方案——所有玩家从同一个天裂形成的裂谷小镇开始，各出身都有合理汇聚理由，且从此向外辐射到各区域。

### 参考资料

- 金庸武侠：门派基于真实中国地理分布（少林/武当/峨眉/桃花岛等），正邪对立体系
- 古龙武侠：青龙会等神秘组织，个人传奇与悬疑氛围，门派作为背景
- 逆水寒：北宋末年背景，温瑞安武侠改编，写实武侠风格
- 燕云十六声：五代十国背景，碎片化叙事，涌现式互动，多门派体系
- 太吾绘卷：架空世界，高自由度，随机出身和天赋
- 活侠传：天命盘，命运分支
